## Windows Fundamentals

This section documents my study of Windows fundamentals with a focus on concepts relevant to SOC analysis and system security.

# Focus Areas:

  - Core Windows processes (e.g., LSASS)
  - File systems and permissions (NTFS vs Share)
  - Network fundamentals (SMB)
  - Command-line usage and file operations

# Structure

  - concepts/ → explanations of key topics and system behavior
  - labs/ → hands-on exercises and practice scenarios
  - screenshots/ → supporting visuals from lab environments

# Goal

To build a strong understanding of how Windows systems operate in order to better analyze normal vs. abnormal behavior.
